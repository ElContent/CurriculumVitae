Proyectos

Reunión de Estudiantes de Ingenierías Técnicas y Superiores en Informática (RITSI)

  • Bot de Telegram de RITSI (BOTSI)
    Desarrollé el bot de Telegram de la Asociación, el cual contiene información, así como un acceso rápido al resto de herramientas, entre otros útiles para RITSI. Está desarrollado con Python.
  • Plataforma de Gestión de Puntuaciones de RITSI (GPR)
    Una plataforma cuyo fin consiste en actualizar las puntuaciones de los equipos de las Jornadas de Formación de RITSI. Se expone una API y la consume el bot de Telegram de la Asociación, facilitando mucho el trabajo a las personas encargadas de actualizar las puntuaciones. Se trata de una plataforma web desarrollada por mi en NodeJS a partir de fronts de Coordinaciones anteriores.
  • Plataforma de Gestión de Censos de RITSI (GCR)
    Una plataforma para gestionar el censo de miembros activos de la Asociación. Así como el censo particular de cada evento, zona y comisión temática. Se trata de una plataforma web desarrollada por mi en NodeJS.
